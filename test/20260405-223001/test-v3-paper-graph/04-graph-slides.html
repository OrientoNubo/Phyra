Development Narrative

DUSt3R (CVPR 2024) demonstrated that multi-view 3D geometry can be recovered through direct pointmap regression from image pairs, bypassing traditional feature extraction and matching pipelines. This simplification invited two immediate follow-ups. MASt3R (ECCV 2024) re-introduced learnable local features on top of DUSt3R's pointmap framework, showing that joint training of matching and reconstruction heads yields stronger correspondences. In parallel, VGGT (CVPR 2025) took DUSt3R's core insight further: rather than processing pairs, it designed a geometry transformer that ingests an entire unordered batch and predicts all 3D attributes in a single feed-forward pass.

VGGT's batch architecture, however, cannot handle arbitrarily long video because memory grows quadratically with sequence length. CUT3R (2025) addressed this by introducing a causal, frame-by-frame reconstruction model that processes temporal streams auto-regressively. InfiniteVGGT (2026) then proposed a more direct solution: it retrofits VGGT itself with a KV-cache rolling memory mechanism, achieving streaming capability while preserving VGGT's representation power. By maintaining constant memory usage regardless of sequence length, InfiniteVGGT supersedes CUT3R's approach for open-ended temporal reconstruction.
