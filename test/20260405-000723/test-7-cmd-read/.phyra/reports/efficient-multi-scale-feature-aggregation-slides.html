Problem

Existing multi-scale fusion methods (FPN, BiFPN, NAS-FPN) apply the same cross-scale fusion to every spatial location, regardless of local context.

  • Large objects on a single scale receive unnecessary multi-scale fusion
  • Small objects near scale boundaries may not receive optimal scale combinations
  • Fixed topologies waste computation on regions that do not benefit from cross-scale interaction

The bottleneck is not the fusion topology itself, but the lack of input-dependent routing.

3 / 8

Method: Lightweight Adaptive Feature Aggregation (LAFA)

O_i = sum_{j in S_i} R_ij(P_i) * Align(P_j, P_i)
  • A routing network (GAP + 2 FC layers, reduction ratio r=16) predicts per-scale routing logits
  • Gumbel-Softmax enables differentiable discrete selection during training
  • At inference, only the top-K scales (K=2) are selected per level, zeroing out the rest

This sparse selection reduces FLOPs by ~38% compared to BiFPN while improving accuracy.

Critical Assessment

The claimed "theoretical analysis" (contribution 2) does not appear in the paper. No theorem, lemma, or formal proof is provided.

  • The routing network uses global average pooling, producing per-image (not per-region) routing decisions, despite the paper's per-region framing
  • "Real-time" is benchmarked exclusively on V100; no edge-device or mobile latency results
  • The 40% reduction figure lacks a precise definition (FLOPs vs. connections vs. activations)
  • RT-DETR, the strongest real-time baseline in related work, is absent from the main comparison table
8 / 8

Summary

  • LAFA introduces input-dependent routing to multi-scale feature aggregation, replacing fixed fusion topologies
  • The approach achieves 48.3 mAP / 42 FPS on COCO val2017 (R-50), surpassing BiFPN by +2.2 mAP and +7 FPS
  • Gains generalize across backbones (R-101, Swin-T) and datasets (Objects365)
  • The core design (GAP + FC routing + Gumbel-Softmax + top-K selection) is simple and integrates into existing detectors

Primary value: a clean, practical module that jointly improves accuracy and speed in multi-scale detection. The missing theoretical analysis and per-region routing discrepancy are notable gaps that weaken two of the three claimed contributions.
