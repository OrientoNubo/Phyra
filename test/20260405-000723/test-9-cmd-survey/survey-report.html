Research Landscape

Zero-shot object detection (ZSD) aims to localize and recognize object categories unseen during training. The field has evolved from semantic-embedding methods (2018) through CLIP-based transfer (2021) to modern scaled grounded detection systems (2023+). Current state-of-the-art methods leverage vision-language models pretrained on web-scale data to achieve open-vocabulary detection without category-specific training.

Key Papers

ECCV 2024
Grounding DINO: Marrying DINO with Grounded Pre-Training for Open-Set Object Detection
Tight fusion of Transformer detector with language grounding. 52.5 AP zero-shot on COCO. Feature enhancer + language-guided query selection + cross-modality decoder.
CVPR 2024
YOLO-World: Real-Time Open-Vocabulary Object Detection
RepVL-PAN for vision-language aggregation. 35.4 AP at 52 FPS on LVIS. Prompt-then-detect paradigm with offline vocabulary for efficient deployment.
NeurIPS 2023
OWLv2: Scaling Open-Vocabulary Object Detection
Self-training on 1B+ web examples. LVIS rare classes AP: 31.2% to 44.6% (43% relative gain). Introduced objectness classifier.
ICML 2021
CLIP: Learning Transferable Visual Models From Natural Language Supervision
Contrastive pretraining on 400M image-text pairs. Foundational model enabling zero-shot visual recognition. Backbone for OWL-ViT, OWLv2, and many OVD methods.

Identified Gaps and Future Directions

1. Domain-Specific Evaluation

Most benchmarks use COCO and LVIS. Evaluation on medical, satellite, and industrial domains remains sparse.

2. Small Object Detection

Zero-shot detection of small objects is poorly addressed; VLMs struggle at low resolutions.

3. Compositional Queries

Detecting objects described by compositional attributes remains challenging for current VLMs.

4. Temporal Consistency in Video

Applying zero-shot detection to video with frame-to-frame consistency is underexplored.

5. Edge Deployment

Only YOLO-World targets real-time; mobile and edge hardware deployment remains an open problem.
