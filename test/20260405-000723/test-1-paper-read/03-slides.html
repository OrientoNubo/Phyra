Overview

What This Paper Does

Proposes LAFA (Lightweight Adaptive Feature Aggregation), a module that replaces fixed multi-scale fusion with input-dependent routing. Instead of fusing all scales uniformly, LAFA learns which scale combinations to fuse for each input, using a lightweight routing network and sparse top-K selection at inference.

Core Idea

From Static Fusion to Adaptive Routing

Existing methods (FPN, BiFPN, NAS-FPN) apply the same fusion topology to every spatial location. LAFA argues this is wasteful: large objects need only single-scale features, while small objects near scale boundaries may need non-standard combinations.

O_i = SUM_{j in S_i} R_ij(P_i) * Align(P_j, P_i) where S_i is dynamically selected per input

The routing function R is predicted by GAP + 2 FC layers with Gumbel-Softmax. At inference, only the top-K=2 scales are selected per level.

Critical Issues

Red Flags Identified

1
Missing theoretical analysis. Contribution 2 claims "theoretical analysis showing sparse selection preserves performance" but no theorem, proof, or formal bound appears anywhere in the paper. (Sec 1, Abstract)
2
Per-region vs. per-image mismatch. The paper claims "per-region routing decisions" but the routing network uses global average pooling, which collapses spatial dimensions. The actual routing is per-image, not per-region. (Sec 3.2)
3
FLOPs not directly measured. The 38-40% FLOPs reduction claim is supported only by FPS proxy measurements. FPS and FLOPs have a non-linear relationship. (Abstract, Sec 3.2)
4
Outdated baselines. All comparison methods (FPN 2017, PANet 2018, NAS-FPN 2019, BiFPN 2020) are 6-9 years old. No 2021-2025 methods are benchmarked despite being cited. (Sec 4.1)
Verdict

Overall Assessment

LAFA presents a reasonable idea (adaptive routing for multi-scale fusion) with solid empirical improvements on standard benchmarks. The core experimental results are internally consistent and the ablation confirms the value of both routing and sparsity.

However, two high-severity issues weaken the paper: (1) claiming a theoretical contribution that does not exist, and (2) mischaracterizing per-image routing as per-region routing. The outdated baselines further limit the impact claim. A revision addressing these gaps would strengthen the submission considerably.

Worth Reading If

  • You work on efficient multi-scale feature fusion for detection
  • You are exploring input-dependent / dynamic network architectures
